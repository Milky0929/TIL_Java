#자바 개요


##자바
- C++과 달리 처음부터 객체 지향 언어로 개발됨
- JVM (자바 가상 머신)을 사용 == **어느 운영체제에서나 같은 형태**로 실행

<br><br>

##자바의 역사
- made by 썬 마이크로 시스템즈, 제임스 고슬링 팀
- 1991 : 오크
- 1996 : 자바 (1.02)
- 1998 : 자바 애플릿 추가 (J2SE1.2), 웹에서 사용 가능
- 2009 : 소유권 -> 오라클

<br><br>

##자바의 장점
- **운영체제와 독립적 실행**
- 불필요한 기능 제거됨
- 자동 메모리 관리 지원 == 안정성 up
- 연산자 오버로딩 금지 == 코드 가독성 up
- 제네릭 도입 == 코드 가독성 up

<br><br>

##자바의 단점
- **가상머신 -> 실행 == 속도 down**
- 일일이 예외 처리 지정해야
- 코드의 길이 long

<br><br>

##자바 표준
- JDK / Java SE
